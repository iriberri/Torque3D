Detailed Description

An async stream queue that writes sound packets to SFXBuffers in sync to the playback of an SFXVoice.

Sound packet queues use sample counts as tick counts.

Member Enumeration Documentation

anonymous enum
Enumerator
DEFAULT_STREAM_QUEUE_LENGTH 

The number of stream packets that the playback queue for streaming sounds will be sliced into. This should generally be left at three since there is an overhead incurred for each additional segment. Having three segments gives one segment for current immediate playback, one segment as intermediate buffer, and one segment for stream writes.
