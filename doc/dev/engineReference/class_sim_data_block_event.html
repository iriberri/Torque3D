Detailed Description

Event for sending a datablock over the net from the server to the client.

Datablock events are GuaranteedOrdered client events.

Member Data Documentation

SimObjectId SimDataBlockEvent::id
protected

Id of the datablock object to be sent. This must be a datablock ID (as opposed to a normal object ID).

protected

The mission sequence number to which this datablock transmission belongs.

U32 SimDataBlockEvent::mTotal
protected

Total number of datablocks that are part of this datablock transmission. Each datablock is transmitted in an independent datablock event.
